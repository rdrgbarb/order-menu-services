# order-menu-services

Two Spring Boot microservices (Menu + Order) for a simple ordering workflow, using MongoDB for persistence and RabbitMQ for status-change notifications.

## Tech stack
- Java (latest)
- Spring Boot
- Maven (multi-module)
- MongoDB
- RabbitMQ (Spring AMQP)
- OpenAPI / Swagger UI (springdoc)
- Docker / docker-compose

---
